1 什么是匿名函数？

就是…没有名字的函数呗，举个例子 

function bob() {
  //...
}

var bob = function() {
  //...
}

前一个函数的名字是bob，后一个就是匿名函数

2 比较

有名的 匿名的
调试方便 立即执行的函数可用于创建一个自己的scope，此时并不需要名字
递归，有名字的函数可以自己调用自己 简洁，如果是一个很短的函数或是回调，匿名函数会很简洁

3 立即执行函数（iffe）

(function() {
  alert('i am a iffe')    
}())

function最外面要加括号，感觉就像是 

var a = function() {
  alert('i am not a iffe')
}
a()

iffe很有用的地方就是，它创造了一个自己的scope，内部的变量不受到外界的影响，也不会污染其他的全局内容 所以JQuery源码上来就是 

JQuery = (function(){
  //...  
}())
